## 💻 Learning Method

For every topic, I follow the same approach:

1. Learn the core concepts.
2. Practice with small examples.
3. Solve multiple exercises.
4. Build a mini project.
5. Refactor the code for readability.
6. Document the project with a README.
7. Commit the work to GitHub before moving to the next topic.

This ensures I understand each concept before progressing.
